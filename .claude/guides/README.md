## 🎯 クイックリファレンス

### マルチエージェントを使うべき？

以下の**すべて**に該当する場合、マルチエージェント推奨：

- [ ] タスクが複数のサブタスクに分割可能
- [ ] 各サブタスクが 5 分以上かかる
- [ ] サブタスクが並行処理可能（依存関係が少ない）
- [ ] 各サブタスクが異なる専門性を持つ

→ **すべて Yes なら [commander.md](commander.md) を読む**

### 使用例

#### ✅ マルチエージェント推奨

- Three.js ページ作成 + 最適化 + テスト
- コードレビュー + バグ修正 + ドキュメント更新
- 大規模リファクタリング + テスト更新

#### ❌ 単独作業推奨

- 単一ファイルの小規模編集（5 分以内）
- 簡単な質問への回答
- 依存関係が強いタスク
